# THEPLUG

The Plug is a static landing page for a home-services brand inspired by the structure of marketplace sites like Taskrabbit. The current version focuses on a clean light blue and white visual style, service discovery, trust-building sections, and clear booking calls to action.

## Project Overview

This repo currently contains a single responsive homepage built with plain HTML and CSS.

The page includes:

- a sticky navigation bar
- a hero section with booking-focused messaging
- service category cards
- social proof and stats
- popular project pricing cards
- testimonials
- trust and guarantee messaging
- a simple how-it-works section
- a footer with service and company links

## File Structure

- `home.html` - main homepage markup
- `styles.css` - standalone stylesheet for layout, colors, spacing, and responsive behavior
- `README.md` - project documentation

## Design Direction

The current visual system uses:

- light blue accents
- white cards and surfaces
- soft gradients
- rounded containers
- responsive grid layouts for desktop and mobile

## How To Run

Because this is a static site, no build tools or dependencies are required.

1. Open `home.html` directly in a browser.
2. Or serve the folder with any simple local server if you want a cleaner preview workflow.

## Customization

You can update the page quickly by editing:

- `home.html` for content, sections, text, and links
- `styles.css` for colors, spacing, typography, and responsive layout

Useful places to customize next:

- replace placeholder service details with your real offerings
- add real images or icons
- connect the booking buttons to a live form or booking page
- rename `home.html` to `index.html` if you want it to be the default site entry file
